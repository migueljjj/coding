  • HTML5 is the latest version of HTML, the standard markup language for creating and structuring web content. It introduces new elements and APIs for enhanced multimedia, graphics, and functionality

  • Three.js is a JavaScript library that simplifies the creation of 3D graphics in the browser, making it easier to build immersive and interactive 3D experiences.

  • PHP is a server-side scripting language commonly used for web development. It facilitates the creation of dynamic web pages and interacts with databases

  • jQuery is a fast and lightweight JavaScript library that simplifies HTML document traversal and manipulation, event handling, and animation.

  • Angular is a front-end framework developed by Google, designed for building dynamic single-page web applications. It uses TypeScript for building robust and scalable applications

  • Vue.js is a progressive JavaScript framework for building user interfaces. It's known for its simplicity and flexibility, making it easy to integrate into other projects.
